What Is Shared Across CLL Omics Layers?

The question

The CLL cohort asks a familiar multi-omics question: when transcript abundance, DNA methylation, and drug response are measured on overlapping patients, which axes of variation are genuinely shared? This vignette applies rajiveplus to the public Chronic Lymphocytic Leukaemia cohort from Dietrich et al. (2018) (PMID 29227291), distributed through BloodCancerMultiOmics2017. The same data were used in the original MOFA2 vignette (Argelaguet et al. 2020), giving a useful point of comparison.

We use three approximately Gaussian views:

Block Source object Biological meaning
mRNA dds (DESeqDataSet, 63 677 genes × 136 samples) Transcriptional programs
Methylation methData (RangedSummarizedExperiment, 5 000 CpGs × 196 samples) Epigenetic state
Drug viability lpdAll (ExpressionSet, 448 drug–dose features × 249 samples) Pharmacological sensitivity

Why no mutations block? The MOFA2 tutorial includes somatic mutations as a fourth block using a Bernoulli likelihood. RaJIVE inherits the JIVE Gaussian assumption, so binary mutation matrices are not used as input here. IGHV mutation status and trisomy 12 instead enter as metadata for interpretation and clinical validation.

The three-way sample overlap after restricting to CLL patients is ~121 samples.

The analysis follows five questions:

  1. Which joint sources of variation are shared across all three layers?
  2. Which feature weights make those axes biologically legible?
  3. Do joint factors align with IGHV status or trisomy 12?
  4. Which features are significant under jackstraw_rajive()?
  5. Can joint factors predict missing IGHV status and time to treatment?

Kaplan-Meier curve (optimal cutpoint)

Instead of splitting at the median, we use the maximally selected rank statistic (surv_cutpoint() from survminer) to choose the Factor 1 threshold. This matches the approach used in the MOFA2 CLL vignette.

if (requireNamespace("survminer", quietly = TRUE) &&
    requireNamespace("survival",  quietly = TRUE) &&
    exists("surv_df") && nrow(surv_df) > 10) {

  # Optimal cutpoint via maximally selected rank statistic
  cut_res <- survminer::surv_cutpoint(
    data      = surv_df,
    time      = "TTT",
    event     = "treatedAfter",
    variables = "Factor1"
  )
  opt_cut <- cut_res$cutpoint$cutpoint
  surv_df$Factor1_group <- ifelse(surv_df$Factor1 > opt_cut, "High", "Low")

  km_fit <- survival::survfit(
    survival::Surv(TTT, treatedAfter) ~ Factor1_group, data = surv_df

Methodological note. MOFA2 is a probabilistic latent factor model with view-specific likelihoods; RaJIVE is a deterministic matrix decomposition that maximizes explained variance under a Gaussian assumption. Both methods identify IGHV status as a dominant CLL axis. The differences matter: RaJIVE gives a hard joint/individual split, its jackstraw test gives feature-level significance without a sparse prior, and MOFA2 natively handles non-Gaussian views such as binary mutations and counts.
