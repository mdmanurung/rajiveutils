1 The question

A gut microbiome is not one ecosystem measured three times. Bacteria, fungi, and viruses can move together, or they can carry kingdom-specific variation. This vignette uses rajiveplus on the multi-kingdom gut microbiome data from Haak et al. (2021) to separate those two possibilities.

The cohort contains 57 patients with matched measurements across three microbial kingdoms - Bacteria (16S rRNA, 180 OTUs), Fungi (ITS2, 18 OTUs), and Viruses (virome sequencing, 42 viral families) - plus clinical metadata including antibiotic exposure and short-chain fatty acid profiles.

Kingdom Technology Features
Bacteria 16S rRNA 180 OTUs
Fungi ITS2 18 OTUs
Viruses virome-seq 42 vOTUs

1.1 Scientific question

Can RaJIVE find a shared microbiome axis across kingdoms, and can it separate that axis from bacteria-, fungi-, or virus-specific structure?

14 Conclusion

In this cached analysis, RaJIVE finds one joint component shared by Bacteria, Fungi, and Viruses, plus kingdom-specific individual components. The joint axis is the candidate ecosystem-wide signal: a host or treatment-associated pattern that crosses kingdom boundaries. The individual components capture the other half of the story, where each kingdom carries its own ecological structure.

The main takeaways are:

  1. Bacteria, Fungi, and Viruses share a single covariation axis (jr = 1)
  2. Jackstraw testing highlights features supporting that joint structure
  3. The joint component can be screened against clinical categories
  4. Individual components retain kingdom-specific biology that the joint axis intentionally leaves behind

16 Appendix: Cache generation script

The code below recreates the cached RDS files from the source data. It needs internet access to download from EBI FTP, so it is not evaluated during vignette builds.

# ---- 0. Setup ----------------------------------------------------------------
library(rajiveplus)
library(dplyr)
library(tidyr)

out_dir <- "vignettes/data"
dir.create(out_dir, recursive = TRUE, showWarnings = FALSE)

# ---- 1. Download raw data from EBI FTP -------------------------------------
message("[1/6] Downloading raw data from EBI FTP...")
base_url <- paste0(
  "https://ftp.ebi.ac.uk/biostudies/nfs/S-BSST666/",
  "MOFA_tutorial_microbiome/")

kingdoms <- c(Bacteria = "16S_otu_abundances.csv",
              Fungi    = "ITS_otu_abundances.csv",
              Viruses  = "virome_otu_abundances.csv")
meta_file <- "sample_metadata.csv"

read_ebi <- function(file) {
  read.csv(url(paste0(base_url, file)), row.names = 1, check.names = FALSE)
}

raw_mats <- lapply(kingdoms, read_ebi)
metadata <- read_ebi(meta_file)

# CLR-normalise (add pseudo-count)
clr_norm <- function(X) {
  X <- X + 0.5
  log(X) - rowMeans(log(X))
}
raw_mats <- lapply(raw_mats, clr_norm)

# Convert to long format for storage
dt_long <- do.call(rbind, lapply(names(raw_mats), function(k) {
  m <- raw_mats[[k]]
  df <- as.data.frame(as.table(m))
  colnames(df) <- c("sample", "feature", "value")
  df$view <- k
  df[, c("sample", "feature", "view", "value")]
}))

raw <- list(dt = dt_long, metadata = metadata)
saveRDS(raw, file.path(out_dir, "microbiome_raw.rds"))
message("    -> saved microbiome_raw.rds")

# ---- 2. Preprocessing -------------------------------------------------------
message("[2/6] Preprocessing to matrix blocks...")
common_samples <- Reduce(intersect, lapply(raw_mats, rownames))
blocks <- lapply(raw_mats, function(m) m[common_samples, , drop = FALSE])
meta_s <- metadata[match(common_samples, rownames(metadata)), ]
rownames(meta_s) <- common_samples

message("    Views found: ", paste(names(blocks), collapse = ", "))
message("    Samples in common: ", length(common_samples))

pp <- list(blocks = blocks, meta = meta_s)
saveRDS(pp, file.path(out_dir, "microbiome_preprocessed.rds"))
message("    -> saved microbiome_preprocessed.rds")
message("    Block dims: ",
        paste(sapply(blocks, function(m) paste(dim(m), collapse = "x")),
              collapse = ", "))

# ---- 3. Per-block SVD for scree plots ---------------------------------------
message("[3/6] Computing per-block SVD for scree plots...")
svd_list <- lapply(blocks, function(m) {
  s <- svd(scale(m), nu = 0, nv = 0)$d
  s
})
saveRDS(svd_list, file.path(out_dir, "microbiome_svd_list.rds"))

init_ranks <- c(Bacteria = 5L, Fungi = 4L, Viruses = 3L)
message("    Initial signal ranks: ",
        paste(names(init_ranks), init_ranks, sep = "=", collapse = ", "))
message("    -> saved microbiome_svd_list.rds")

# ---- 4. Fit RaJIVE ----------------------------------------------------------
message("[4/6] Fitting Rajive() model...")
mb_fit <- Rajive(blocks, initial_signal_ranks = init_ranks)
jr <- get_joint_rank(mb_fit)
message("    Estimated joint rank: ", jr)
saveRDS(mb_fit, file.path(out_dir, "microbiome_rajive_results.rds"))
message("    -> saved microbiome_rajive_results.rds")

# ---- 5. Jackstraw -----------------------------------------------------------
message("[5/6] Running jackstraw_rajive()...")
js <- jackstraw_rajive(mb_fit, blocks = blocks, n_null = 100L)
saveRDS(js, file.path(out_dir, "microbiome_jackstraw_results.rds"))
message("    -> saved microbiome_jackstraw_results.rds")

# ---- 6. Bootstrap stability -------------------------------------------------
message("[6/6] Bootstrap rank stability (B=30)...")
stab <- assess_stability(
  ajive_output        = mb_fit,
  blocks              = blocks,
  initial_signal_ranks = init_ranks,
  target              = "joint_rank",
  B                   = 30L,
  sample_frac         = 0.8)
saveRDS(stab, file.path(out_dir, "microbiome_stability.rds"))
message("    -> saved microbiome_stability.rds")
message("All 6 cache files generated successfully.")
